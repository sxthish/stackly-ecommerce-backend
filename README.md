# 🛒 STACKLY - E-Commerce Backend

A complete E-Commerce REST API built using Spring Boot.

## Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Lombok
- Postman


## Features

### User Management
- User Registration
- Role Management
- Address Management


### Product Management
- Category Management
- Product CRUD
- Inventory Management
- Search Products
- Pagination & Sorting


### Cart System
- Add Product to Cart
- Remove Items
- Calculate Total Amount


### Order Management
- Place Order
- Order Tracking
- Order Status


### Payment System
- Payment Processing
- Transaction ID Generation


### Shipment
- Shipment Creation
- Tracking Number
- Delivery Status


### Review System
- Product Rating
- Customer Reviews


## Architecture

Controller
    ↓
Service
    ↓
Repository
    ↓
Database


## Database

MySQL with Hibernate ORM

Entities:

User

Product
Category
Inventory
Cart
